🔄 氣泡排序法

Bubble Sort - 最簡單直觀的排序演算法

📚 演算法原理

什麼是氣泡排序？

氣泡排序是一種簡單的排序演算法，它重複地走訪要排序的數列，一次比較兩個元素，如果順序錯誤就交換。這個過程不斷重複，直到沒有需要交換的元素為止。

之所以稱為「氣泡」排序，是因為較小的元素會像氣泡一樣「浮」到數列的前面，而較大的元素會「沈」到後面。

基本流程

  1. 比較相鄰的兩個元素
  2. 如果前一個元素大於後一個元素，就交換它們
  3. 對每一對相鄰元素重複第 1-2 步
  4. 第一輪後，最大的元素會移到末尾
  5. 重複上述過程，直到整個數列排序完成

Python 實現

def bubble_sort(arr): n = len(arr) # 外層迴圈：控制比較的輪數 for i in range(n): swapped = False # 內層迴圈：每輪比較相鄰的元素 for j in range(0, n - i - 1): if arr[j] > arr[j + 1]: # 交換元素 arr[j], arr[j + 1] = arr[j + 1], arr[j] swapped = True # 如果沒有發生交換，說明已排序完成 if not swapped: break return arr # 使用範例 numbers = [64, 34, 25, 12, 22, 11, 90] result = bubble_sort(numbers) print(result) # 輸出: [11, 12, 22, 25, 34, 64, 90]

C 語言實現

#include <stdio.h> void bubble_sort(int arr[], int n) { // 外層迴圈：控制比較的輪數 for (int i = 0; i < n; i++) { int swapped = 0; // 內層迴圈：每輪比較相鄰的元素 for (int j = 0; j < n - i - 1; j++) { if (arr[j] > arr[j + 1]) { // 交換元素 int temp = arr[j]; arr[j] = arr[j + 1]; arr[j + 1] = temp; swapped = 1; } } // 如果沒有發生交換，說明已排序完成 if (!swapped) break; } } int main() { int arr[] = {64, 34, 25, 12, 22, 11, 90}; int n = sizeof(arr) / sizeof(arr[0]); bubble_sort(arr, n); printf("排序後: "); for (int i = 0; i < n; i++) printf("%d ", arr[i]); return 0; }

📊 複雜度分析

指標 最好情況 平均情況 最壞情況
時間複雜度 O(n) O(n²) O(n²)
空間複雜度 O(1) - 原地排序
穩定性 ✓ 穩定 - 相等元素的相對順序不變

複雜度說明

最好情況 O(n)：當陣列已經有序時，只需一次遍歷確認即可（使用優化的bubble_sort）

平均/最壞情況 O(n²)：需要 n 輪比較，每輪約 n 次操作，導致 n × n = n² 次操作

空間複雜度 O(1)：只需要常數額外空間，不需要額外的陣列

⚖️ 優缺點對比

✓ 優點

易於理解

演算法邏輯簡單直觀，適合初學者學習

原地排序

不需要額外的記憶體空間

穩定

相等元素的相對位置不會改變

自適應

對已排序或近似排序的陣列效率較高

✗ 缺點

效率低下

時間複雜度為 O(n²)，不適合大規模資料

比較次數多

即使資料已排序也要進行多次比較

交換頻繁

涉及大量的元素移動操作

不適合實際應用

實際應用中通常使用更高效的演算法

🎯 應用場景

教育目的

作為排序演算法的入門教學，幫助學生理解排序的基本概念

小規模資料

對於只有幾十個元素的小陣列，氣泡排序足夠使用

幾乎有序的資料

對於已經接近排序的陣列，優化後的氣泡排序表現良好

嵌入式系統

在記憶體限制的環境中，原地排序的特性很有價值
